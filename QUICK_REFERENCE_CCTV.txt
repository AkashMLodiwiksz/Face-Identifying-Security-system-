┌─────────────────────────────────────────────────────────────────────┐
│  HOW TO ADD A CAMERA (5 STEPS)                                      │
├─────────────────────────────────────────────────────────────────────┤
│  1. Login to your account                                           │
│  2. Click "Cameras" in sidebar                                      │
│  3. Click "Add Camera" button                                       │
│  4. Fill in form:                                                   │
│     - Name: "Front Door Camera"                                     │
│     - RTSP URL: rtsp://admin:password@192.168.1.100:554/stream     │
│  5. Click "Add Camera" → Then "Test" button                         │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│  RTSP URL FORMAT                                                    │
├─────────────────────────────────────────────────────────────────────┤
│  rtsp://[username]:[password]@[ip]:[port]/[path]                    │
│                                                                     │
│  Example:                                                           │
│  rtsp://admin:password123@192.168.1.100:554/stream1                │
└─────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────┐
│  COMMON CAMERA URLS                                                 │
├─────────────────────────────────────────────────────────────────────┤

│  rtsp://admin:pass@192.168.1.103:554/stream1                       │
│                                                                     │

│  rtsp://admin:pass@192.168.1.104:554/live                          │

┌─────────────────────────────────────────────────────────────────────┐
│  TEST RTSP URL WITH VLC                                             │
├─────────────────────────────────────────────────────────────────────┤
│  1. Open VLC Media Player                                           │
│  2. Press Ctrl+N (or Media → Open Network Stream)                   │
│  3. Paste your RTSP URL                                             │
│  4. Click Play                                                      │
│  5. ✅ Video shows = URL is correct!                                │
└─────────────────────────────────────────────────────────────────────┘
